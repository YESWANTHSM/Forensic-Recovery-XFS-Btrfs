Day 14 – Recovery Tools Study
Introduction

File recovery tools are essential in digital forensics for recovering deleted or lost files. In this project, TestDisk and PhotoRec are studied because they are widely used open-source recovery tools.

1. TestDisk
Overview

TestDisk is a free and open-source data recovery tool used to recover lost partitions and repair damaged file systems.

Features
Partition recovery
Boot sector repair
File system recovery
Supports Linux file systems
Advantages
Easy to use
Reliable recovery
Open source
2. PhotoRec
Overview

PhotoRec is a file recovery utility that recovers deleted files from storage devices regardless of the file system.

Features
Recovers deleted files
Supports multiple file formats
Works with damaged file systems
Advantages
High recovery success rate
Supports many file types
Free and open source
Comparison of Recovery Tools
Feature	TestDisk	PhotoRec
Partition Recovery	Yes	No
File Recovery	Yes	Yes
File System Repair	Yes	No
Open Source	Yes	Yes
Relevance to the Project

These tools will be used to:

Recover deleted files from XFS and Btrfs.
Analyze recovered data.
Compare recovery effectiveness.
Expected Outcome

The study of TestDisk and PhotoRec provides the necessary knowledge required for performing deleted file recovery experiments in the project.

Outcome

Recovery tools required for deleted file recovery and forensic analysis were studied successfully.
