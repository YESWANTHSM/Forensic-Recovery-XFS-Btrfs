# Day 6 – Software and Hardware Requirements

## Introduction

The successful implementation of the project requires suitable hardware and software resources. This document identifies the necessary requirements for developing and testing the forensic recovery system.

## Hardware Requirements

* Processor: Intel Core i3/i5 or higher
* RAM: Minimum 8 GB
* Storage: 50 GB free disk space
* Internet Connection: Required for downloading tools and updates

## Software Requirements

* Windows 10/11 Operating System
* Oracle VirtualBox
* Ubuntu Linux
* Python Programming Language
* Git and GitHub
* TestDisk Recovery Tool
* PhotoRec Recovery Tool

## Purpose of Requirements

These tools and resources are required to create XFS and Btrfs file systems, perform file deletion experiments, recover deleted files, and analyze metadata.

## Outcome

The hardware and software requirements for the project were identified and documented successfully.
