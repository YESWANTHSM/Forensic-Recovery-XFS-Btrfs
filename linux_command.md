Day 9 – Linux Commands Study
Introduction

Linux commands are used to interact with the operating system through the terminal. Learning basic commands is essential for managing files, directories, and system operations required for this project.

Commands Studied
1. pwd

Displays the current working directory.

pwd
2. ls

Lists files and directories.

ls
3. cd

Changes the current directory.

cd Documents
4. mkdir

Creates a new directory.

mkdir ProjectFiles
5. rm

Removes files or directories.

rm file.txt
6. cp

Copies files from one location to another.

cp file.txt backup.txt
7. mv

Moves or renames files.

mv file.txt newfile.txt
8. cat

Displays the contents of a file.

cat file.txt
Importance in the Project

These commands will be used to create test files, manage directories, delete files, and perform recovery experiments on XFS and Btrfs file systems.

Outcome

Basic Linux commands were successfully studied and practiced in the Ubuntu environment.
