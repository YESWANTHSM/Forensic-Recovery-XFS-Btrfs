# Forensic Recovery and Analysis of Deleted Data in XFS and Btrfs File Systems

## Objective
To recover deleted files and analyze metadata from XFS and Btrfs file systems.

## Tools Used
- Ubuntu
- Python
- TestDisk
- PhotoRec

## Project Workflow
1. Create XFS and Btrfs file systems
2. Store sample files
3. Delete files
4. Recover deleted files
5. Analyze metadata
6. Compare results
